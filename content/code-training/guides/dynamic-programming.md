# 动态规划

## 适用信号

- 最优值、方案数、可行性
- 子问题有重叠
- 当前状态可由更小状态转移得到

## 固定拆解

1. 定义状态：`dp[i]` 或 `dp[i][j]` 表示什么。
2. 写转移：当前状态从哪些旧状态来。
3. 初始化：最小子问题是什么。
4. 遍历顺序：保证依赖状态已经算好。
5. 返回答案：返回哪个状态。

## 一维 DP 框架

```python
dp = [initial] * (n + 1)
dp[0] = base

for i in range(1, n + 1):
    for choice in choices:
        dp[i] = update(dp[i], dp[previous_state])
```

## 常见坑

- 状态定义含糊
- 初始化和返回值不匹配
- 01 背包和完全背包遍历方向混淆
